> The web is a sea of information. Everyone deserves their own pond.

## Getting Started

Download the latest release.

```bash
brew install --cask pond
```

Or grab a `.dmg` from [Releases](https://github.com/raphaelsalaja/pond/releases).

## Development

```bash
pnpm install
pnpm dev
```
